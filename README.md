# TODO APP (current name lol)

My very first project written in C#! A TODO application to give you an overview of your daily tasks! 
Still under construction...

## Prototype features !
- [x] Adding tasks to a ToDo List
- [x] Create a description of a task
- [x] Displaying active tasks
- [x] Ability to add properties to a task
  - [x] Deadlines
  - [ ] People
  - [ ] Place
  - [ ] Priority
- [x] Marking a task as done
- [ ] Displaying finished tasks
